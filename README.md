### Требования к серверу
Apache_2.4 PHP_8.0-8.1 
PHP 8.1
MySQL-8.0

### Старт проекта
Создать базу данных laravel_api
Импортировать базу данных данные в из файла laravel_api.sql
В настройках сервена Домены имя домена (имя папки проекта), Папка домена (\имя папки проекта\public)

### Работа
Зарегестрировать пользователя в вкладке Sign up
Зайти в личный кабинет и создать обьявления квартиры или машины
После одобрения модератором (администратором оно будет) размещено на доско обьявлений
Вход в админку admin@mail.ru пароль 123
Администратор может одобрить обьявление или его удалить, также может создать категорию машин (для примера) при расширении кода может создавать и другие категории
